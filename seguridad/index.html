Uso responsable

Seguridad

Criterios para identificar fuentes oficiales, utilizar herramientas del sistema y comunicar vulnerabilidades.

Fuentes oficiales

El código oficial de Switch Turbo Monitor se publica únicamente en PunchiSoft/switch-turbo-monitor. Compruebe la dirección antes de descargar, clonar o seguir instrucciones.

Dependencias y permisos

Revise las dependencias, mantenga Fedora actualizado y evite ejecutar scripts que no comprenda. Switch Turbo Monitor solicita privilegios mediante PolicyKit solo cuando una operación los necesita; no ejecute toda la aplicación como administrador.

Cambios del sistema

Modificar Turbo Boost o perfiles de energía puede cambiar temperatura, consumo y rendimiento. Verifique la compatibilidad del equipo, cierre tareas importantes y conozca el procedimiento para volver a un perfil equilibrado.

Divulgación responsable

¿Encontraste una vulnerabilidad?

Utiliza la pestaña Security del repositorio afectado y la opción Report a vulnerability, si está disponible. No publiques contraseñas, datos personales, pruebas de concepto explotables ni detalles que puedan poner a otros usuarios en riesgo.

Si el reporte privado no está habilitado, abre una incidencia mínima solicitando un canal privado, sin revelar información sensible.
