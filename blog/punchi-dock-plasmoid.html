Punchi Dock 0.2.0: un dock ligero para organizar lanzadores en Plasma 6

Un plasmoide para KDE Plasma 6 que reúne aplicaciones, carpetas, separadores y módulos rápidos en un dock visual, configurable y pensado para convivir con el gestor de tareas de Plasma.

Punchi Dock nace como una respuesta concreta a una necesidad simple: tener un dock de lanzadores en Plasma 6 que sea cómodo, visual y fácil de ajustar sin convertirlo en un segundo gestor de ventanas. Su lugar natural está entre los accesos frecuentes y las pequeñas herramientas de escritorio, no en la administración de tareas abiertas.

Un dock para lanzar y ordenar

El proyecto está pensado para abrir aplicaciones, agrupar comandos en carpetas, separar zonas visuales y mantener algunos módulos internos a mano. Puede mostrar reloj, calendario y Papelera, además de lanzadores con nombre, icono y comando personalizado.

La administración de ventanas sigue siendo tarea del gestor de tareas de KDE Plasma. Esa decisión mantiene el alcance claro: Punchi Dock no minimiza, restaura, enfoca, fija ni agrupa ventanas abiertas. Tampoco modifica la configuración global de Plasma ni toca plasma-org.kde.plasma.desktop-appletsrc.

La idea no es reemplazar Plasma, sino sumar un dock de lanzadores que se sienta nativo dentro de Plasma.

Qué puede hacer

La versión 0.2.0 permite crear una barra con aplicaciones, carpetas de aplicaciones, separadores, espaciadores, reloj, calendario y Papelera. Las carpetas pueden abrirse como popup en cuadrícula o lista, con opciones para columnas, filas, tamaño de celda, etiquetas y cierre automático después de lanzar una app.

También incluye indicadores simples de ejecución para lanzadores configurados. Son útiles para casos comunes, porque revisan el nombre de proceso inferido desde el comando, pero no buscan competir con el seguimiento completo del gestor de tareas de KDE.

Stack principal

KDE Plasma 6 · QML · JavaScript · kpackagetool6 · GPL-3.0-or-later

Configuración visual

Punchi Dock permite ajustar tamaño de iconos, separación, zoom al pasar el cursor, largo máximo, transparencia, radio de esquinas, orientación, etiquetas y fuente del reloj. La orientación puede ser horizontal, vertical o automática, de modo que el dock se adapta mejor al panel o al borde donde se ubique.

El dock se dimensiona según la cantidad de elementos. Si se define un largo máximo y la lista crece demasiado, el contenido se vuelve desplazable en vez de estirar indefinidamente el marco visual.

Items: edición simple y modo avanzado

La forma recomendada de editar el dock es abrir la configuración del plasmoide y usar la página Items. Desde ahí se pueden agregar aplicaciones, carpetas, separadores, espaciadores, reloj, calendario y Papelera sin editar archivos manualmente.

Para usuarios técnicos, la vista Advanced JSON permite revisar, copiar o modificar la configuración completa. Internamente, Punchi Dock guarda sus elementos en:

~/.config/punchi-dock/dock_items.json

Si el archivo no existe, el plasmoide crea una configuración inicial con Firefox, Home, Konsole, carpetas de desarrollo, oficina y sistema, además de Papelera, separadores, reloj y calendario.

Carpetas y módulos internos

Las carpetas son una de las partes más útiles del proyecto. Permiten agrupar aplicaciones por contexto, como desarrollo, oficina o sistema, y mostrarlas en una cuadrícula compacta o una lista más tradicional. Cada app dentro de la carpeta conserva su propio nombre, icono y comando.

El proyecto evita cargar plasmoides externos dentro del dock. En su lugar usa módulos internos como clock, calendar, trash, separator y spacer. Esto reduce complejidad y mantiene el comportamiento bajo control dentro del propio paquete.

Instalación local y empaquetado

Punchi Dock se instala para el usuario actual mediante kpackagetool6, sin usar sudo. Desde la carpeta del proyecto, la instalación o actualización local se realiza con:

git clone https://github.com/PunchiSoft/punchi-dock-plasmoid.git
cd punchi-dock-plasmoid
chmod +x install-plasmoid.sh
./install-plasmoid.sh

También se puede generar un paquete instalable .plasmoid:

chmod +x build-plasmoid.sh
./build-plasmoid.sh

El paquete queda disponible en dist/punchi-dock.plasmoid. El ID del plasmoide es org.kde.plasma.punchi-dock, la versión documentada aquí es 0.2.0 y el mínimo declarado para Plasma es 6.0.

Compatibilidad y límites claros

El plasmoide está orientado a KDE Plasma 6 y funciona en sesiones Wayland y X11. No depende de herramientas como xdotool, wmctrl o xprop, precisamente porque no intenta manipular ventanas abiertas.

Ese límite también lo vuelve más predecible: Punchi Dock organiza accesos, carpetas y módulos rápidos; KDE Plasma sigue encargado de las ventanas, tareas y comportamiento global del escritorio.

Para quién es

Punchi Dock encaja bien si quieres un dock visual para organizar accesos frecuentes, separar grupos de trabajo y mantener a mano pequeños módulos del escritorio. También sirve como base de aprendizaje para estudiar plasmoides modernos: QML, configuración de widgets, archivos JSON de usuario y empaquetado .plasmoid.

Como proyecto PunchiSoft, sigue la misma dirección de otras herramientas del sitio: hacer más comprensibles tareas técnicas del escritorio Linux, con una interfaz concreta y un alcance honesto.
